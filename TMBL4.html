Network models

Network models are important for several reasons:

  1. Interesting to study in their own right

  2. Provide null models with which to compare results for real-world networks - a bit like the normal distribution in statistics

  3. Study the effect of network structure on dynamics (synchrony in neural networks for example)

Erdos-Renyi random graphs

Here we consider Erdos-Renyi (ER) random graphs, which are denoted by $G(n,p)$

Here, $G(n,p)$ is not a single graph but rather it is a collection of graphs

\[ \begin{align*} n &- \text{ number of nodes}\\ p &- \text{ probability that any 2 nodes are connected} \end{align*} \]

To construct an ER network we start off with a set of $n$ disconnected nodes and connect pairs of nodes independently with probability $p$

As noted overpage, $G(n, p)$ denotes an ensemble of networks

Below we list some of the most important properties of ER networks

  1. The expected number of edges:

    \[\langle m\rangle = \frac{n(n-1)p}{2}\]

  2. The expected degree: $\qquad\displaystyle \langle k\rangle = (n-1)p$

  3. The expected WS clustering coefficient: $\qquad\displaystyle \langle C\rangle = p$

Note: The third result follows since

\[ \frac{\# \Delta\text{s}}{\# 2-\text{paths}} \]

is just the probability that a node's neighbours are connected

We shall return to these properties later

As mentioned previously, ER networks are commonly used as null models

For example, given a network $N=(V, E)$ such that $|V|=n$ we would say that the WS clustering is 'large' if

\[ \color{red}{\boxed{\color{white}{ \frac{\langle C\rangle}{\langle C_\mathrm{rand}\rangle} >> 1 }}} \]

Here $\displaystyle \langle C_\mathrm{rand}\rangle$ denotes the WS clustering coefficient averaged over a large number of ER networks with comparable statistics (i.e. $n$ nodes etc.)

In other words the real-world network has WS clustering much greater than expected

Edge and degree distributions for ER networks

Note that each network in $G(n,p)$ occurs with probability

\[ P(G(n,p)) = p^m\left(1-p\right)^{\frac{n(n-1)}{2}-m} \qquad\left(\frac{n(n-1)}{2} \text{ is the number of links}\right) \]

The probability of an ER network having $m$ edges is

\[ \color{red}{\boxed{\color{white}{ P(m) = \binom{\frac{n(n-1)}{2}}{m}p^m\left(1-p\right)^{\frac{n(n-1)}{2}-m} }}} \]

The number of edges in an ER network follows a binomial distribution

Property (1) of ER networks follows immediately from the above

Similar arguments show

\[ \color{red}{\boxed{\color{white}{ P(k) = \binom{n-1}{k}p^k\left(1-p\right)^{n-1-k} }}} \]

so that the degree distribution of an ER network follows a binomial distribution

Since

\[ k \sim\mathrm{Bi}(n-1,p)\implies \langle k\rangle = (n-1)p \]

i.e. property (2)

Now recall that for large $n$ and small $p$ the binomial distribution is well approximated by the Poisson distribution ...

To see this recall

\[ \begin{align*} \binom{n-1}{k}p^k\left(1-p\right)^{n-1-k} &=\frac{(n-1)(n-2)\cdots(n-k)}{k!}\left(\frac{\langle k\rangle}{n-1}\right)^k\left(1-\frac{\langle k\rangle}{n-1}\right)^{n-1-k}\\ &\approx \frac{\langle k\rangle^k}{k!}\left(1-\frac{\langle k\rangle}{n-1}\right)^{n-1}\\ &\approx \frac{\langle k\rangle^k}{k!}e^{-\langle k\rangle} \end{align*} \]

Thus for large $n$: $\qquad\color{red}{\boxed{\color{white}{\displaystyle k\sim\mathrm{Po}(\langle k\rangle)}}}$

In the above we use the results:

\[ \langle k \rangle = (n-1)p, \qquad n>>k \implies \frac{(n-1)(n-2)\cdots(n-k)}{(n-1)^k}\approx 1 \]

and Euler's definition of the exponential function

\[ e^x = \lim_{n\to\infty} \left(1+\frac{x}{n}\right)^n \]

Python ER networks example

The code describes how to sample and plot an ER network in Python


		import numpy as np
		import networkx as nx
		import matplotlib.pyplot as plt
		from scipy.stats import binom, poisson

		# ----------------------------------------------------------
		# 1. Sample and plot ER networks for p = 0.01, 0.05, 0.1
		# ----------------------------------------------------------

		n = 1000
		p_values = [0.01, 0.05, 0.1]

		plt.figure(figsize=(14, 4))

		for i, p in enumerate(p_values, 1):
			G = nx.erdos_renyi_graph(n, p)
   
			plt.subplot(1, 3, i)
			nx.draw_spring(G, node_size=15,
						node_color="red",
						edge_color="gray", alpha=0.05,
						linewidths=0)
			plt.title(f"ER Graph n={n}, p={p}")

		plt.tight_layout()
		Plt.show()

Python ER networks example (ctd)

		
		n = 1000
		p = 0.1
		lam = n * p

		# Sample the graph
		G = nx.erdos_renyi_graph(n, p)
		degrees = np.array([d for _, d in G.degree()])

		# Range of possible degrees
		k_vals = np.arange(0, 200)

		# Theoretical distributions
		binom_pmf = binom.pmf(k_vals, n-1, p)
		poisson_pmf = poisson.pmf(k_vals, lam)

		plt.figure(figsize=(10, 6))

		# Empirical histogram
		plt.hist(degrees, bins=40, density=True, alpha=0.6, label="Empirical degree distribution")

		# Binomial
		plt.plot(k_vals, binom_pmf, 'r-', lw=2, label="Binomial(n−1, p)")

		# Poisson
		plt.plot(k_vals, poisson_pmf, 'g--', lw=2, label=f"Poisson(λ={lam})")

		plt.xlabel("Degree")
		plt.ylabel("Probability / Density")
		plt.title("Degree Distribution for ER Graph (n=1000, p=0.1)")
		plt.legend()
		plt.show()

Using generating functions to study ER networks

Many properties of random networks can be easily understood via the use of generating functions

Given a discrete probability distribution $P(k)$ its generating function is given by

\[ \color{red}{\boxed{\color{white}{ g(x) = \sum_{k=0}^\infty P(k)x^k }}} \]

For our purposes ...

... the two most important properties of generating functions are the following

\[ \color{red}{\boxed{\color{white}{ \mathbb{E}[n] = \langle n\rangle = \left.\frac{\mathrm{d}g}{\mathrm{d}x}\right|_{x=1} = g'(1) }}} \]

We can use this property to work out the mean properties of a random network

\[ \color{red}{\boxed{\color{white}{ \begin{align*} \mathrm{Var}[n] = \langle (k-\langle k\rangle)^2\rangle &= \left.\frac{\mathrm{d}^2g}{\mathrm{d}x^2}\right|_{x=1} +\left.\frac{\mathrm{d}g}{\mathrm{d}x}\right|_{x=1}\left(1-\left.\frac{\mathrm{d}g}{\mathrm{d}x}\right|_{x=1}\right)\\ & \\&=g''(1)+g'(1)(1-g'(1)) \end{align*} }}} \]

And the standard deviation of these properties

Example 4.2

The degree distribution is

\[ p_0 = 0, ~p_1 = \frac{1}{4}, ~p_2 = \frac{1}{2}, ~p_3 = \frac{1}{4}, ~p_k = 0, ~k\geq 4 \]

Thus the generating function is

\[ g(x)=\frac{x}{4} +\frac{x^2}{2} + \frac{x^3}{4} \]

Note that

\[ \color{red}{\boxed{\color{white}{ g(1) = \frac{1}{4} + \frac{1}{2} + \frac{1}{4} = 1}}} \qquad \color{white}{\left(\sum_i p_i = 1~ \text{i.e. probability distribution}\right)} \] and \[ \begin{align*} g'(x) &= \frac{1}{4} +x + \frac{3}{4}x^2\\ g''(x)&= 1+\frac{3}{2}x \end{align*} \] We need these to work out the mean and variance of the network degree

Example 4.2 (ctd)

Since

\[ \begin{align*} g'(x) &= \frac{1}{4} +x + \frac{3}{4}x^2\\ g''(x)&= 1+\frac{3}{2}x \end{align*} \]

we have that

\[ \langle k\rangle = g'(1) = \frac{1}{4} + 1 + \frac{3}{4} \]

and

\[ \begin{align*} \mathrm{Var}(x) = \sigma^2_k&=g''(1)+g'(1)(1-g'(1))\\ &=1+\frac{3}{2}+2(1-2)\\ &= 0.5 \end{align*} \]
